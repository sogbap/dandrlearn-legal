2. Description du service

D&R Learn est une application mobile d'apprentissage par flashcards permettant :

  • Création et gestion de cours et flashcards personnalisés
  • Sessions de révision en différents modes (swipe, QCM, écriture)
  • Suivi de la progression et statistiques d'apprentissage
  • Génération automatique de questions à choix multiples (distracteurs QCM) lors des révisions
  • Génération automatique de flashcards depuis des documents importés (fonctionnalité Premium)

Note : La synchronisation multi-appareils via iCloud est prévue dans une version ultérieure et n'est pas disponible dans la version actuelle (v1.0).

3. Modèle freemium et abonnements

3.1 Version gratuite

La version gratuite de D&R Learn est accessible sans limitation de cours ni de flashcards. Elle inclut la création de flashcards, les trois modes de révision (swipe, QCM, écriture), les statistiques de progression, et l'intégralité des fonctionnalités de base. Aucune publicité n'est affichée dans la version actuelle (v1.0).

3.2 Abonnement Premium

L'abonnement Premium débloque les fonctionnalités avancées suivantes :

  • Import de documents et génération automatique de flashcards (PDF, Word, PowerPoint, images)
  • Photos dans les flashcards (galerie ou appareil photo)
  • Mise en forme du texte enrichi (gras, italique, couleurs, taille)
  • Filtre de révision par statut des cartes

3.3 Tarifs et renouvellement

Deux formules d'abonnement sont disponibles :

  • Premium Mensuel : 1,99 € par mois, renouvelé automatiquement chaque mois, sans engagement
  • Premium Annuel : 9,99 € par an, incluant 7 jours d'essai gratuit pour les nouveaux abonnés, renouvelé automatiquement chaque année

Les tarifs indiqués sont applicables en zone Euro et peuvent varier selon votre pays et votre devise locale, conformément aux paliers de prix Apple. Le tarif effectif vous est présenté lors de l'achat dans l'Application.

Le renouvellement automatique peut être désactivé à tout moment dans les Réglages de votre iPhone : Réglages → votre nom → Abonnements → D&R Learn. La résiliation prend effet à la fin de la période d'abonnement en cours. Aucun remboursement n'est accordé pour les périodes partiellement utilisées, sauf si la loi applicable l'exige.

3.4 Essai gratuit

L'essai gratuit de 7 jours est disponible uniquement pour les nouveaux abonnés n'ayant jamais souscrit à un abonnement Premium annuel D&R Learn. À l'issue de l'essai, l'abonnement est automatiquement converti en abonnement payant sauf résiliation avant la fin de la période d'essai.

3.5 Restauration des achats

Si vous avez souscrit un abonnement et changez d'appareil ou réinstallez l'Application, vous pouvez restaurer vos achats gratuitement via Profil → Compte → Restaurer les achats, ou via les Réglages Apple ID de votre appareil. Aucun nouveau paiement n'est exigé pour récupérer un abonnement déjà actif lié à votre identifiant Apple.
